채팅 테스트 (프록시 실호출)

⚠ 서버에 실제 OPENAI_API_KEY/ANTHROPIC_API_KEY가 설정돼 있어야 정상 응답이 옵니다. 미설정 시 502(PROVIDER_ERROR)가 정상입니다. 두 번째 호출의 응답 시간이 크게 짧으면 exact 캐시 히트입니다.

레이트리밋 데모 (토큰 버킷)

서버 설정: capacity=10, refillPerSec=1 (10개 버스트 허용 후 초당 1개씩 리필). 11번 연속으로 빠르게 보내면 마지막 요청은 429가 뜨는 것이 정상입니다.
